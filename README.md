## Library Management System

## Aim
To build a Library Management System using React and Next.js that allows users to search, add, and remove books efficiently.

---

## Objectives
- Create a responsive book listing interface  
- Implement real-time search functionality  
- Add a form to insert new book entries  
- Enable removal of books from the list  
- Manage application state using React hooks  

---

## Technologies Used
- **Node.js** (version 18 or above)
- **Next.js** (App Router)
- **React.js**
- **JavaScript**
- **CSS**
- **Vercel** (for deployment)

## How to Run the Project Locally

1. Clone the repository:
   ```bash
   git clone <repository-url>

2. Navigate to the project directory:
   cd library-management-system

3. Install dependencies:
   npm install

4. Start the development server:
   npm run dev

5. Open the browser and visit:
   http://localhost:3000
